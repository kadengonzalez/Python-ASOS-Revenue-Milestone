# ASOS Revenue Analysis Milestone

## Overview
This project focuses on cleaning and analyzing weekly revenue data for ASOS using Python.

The dataset contained a mix of integers and strings, which required cleaning before performing calculations. After cleaning the data, the project analyzes monthly totals and averages.

The goal of this exercise was to practice data cleaning, loops, list slicing, and basic statistical analysis in Python.

---

## Skills & Tools
- Python
- Jupyter Notebook
- Lists
- Loops
- Data cleaning
- Type conversion (`int()`)
- List slicing
- Basic statistics (sum, average)

---

## Key Concepts Practiced
- Converting mixed data types into consistent numeric format
- Using loops to process data
- Appending values to a list
- Slicing lists to separate data by month
- Calculating totals and averages
- Identifying trends in data
